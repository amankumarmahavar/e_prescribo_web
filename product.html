E-Prescription is an Android-based project to provide prescription management and easy report analysis. The main aim of developing this project is to digitalize the prescription system and make it easy to manage. The system provides an interface where users can write prescriptions, edit prescriptions, and share prescriptions with patients. This prescription can also be viewed by the chemist shop if the patient gives permission and directly gives the required medicament.
